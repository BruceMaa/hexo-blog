## Python 科学计算第三方库

### matplotlib

> 绘制一些高质量的的数学二维图形。

```shell
pip install --user matplotlib
```

> 依赖安装的其它库
> 
- pyparsing
- python-dateutil
- cycler
- numpy
- pytz
- six

### scipy

> 基于Python的matlab实现，旨在实现matlab的所有功能。

```shell
pip install --user scipy
```

### numpy

> 基于Python的科学计算第三方库，提供了矩阵，线性代数，傅立叶变换等等的解决方案。

```shell
pip install --user numpy
```
